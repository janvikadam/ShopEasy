🛒 ShopEasy – Full Stack E-Commerce Website

ShopEasy is a full-stack e-commerce web application designed to provide a smooth and user-friendly online shopping experience. The project includes both frontend and backend, supporting essential e-commerce features such as product browsing, user authentication, cart management, and order handling.

📌 Features

🔐 User authentication (Register / Login)

🛍️ Browse products with details

🛒 Add to cart & manage cart items

💳 Order placement functionality

📦 Backend APIs for products, users, and orders

📱 Responsive user interface

⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/your-username/ShopEasy.git
cd ShopEasy

2️⃣ Frontend Setup
cd frontend
npm install
npm start


Frontend will run on:

http://localhost:3000

3️⃣ Backend Setup
cd backend
npm install
node server.js


Backend will run on:

http://localhost:5000

🔐 Environment Variables

Create a .env file inside the backend folder:

PORT=5000
DB_URL=your_database_url
JWT_SECRET=your_secret_key
